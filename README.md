# Отраслевая платежная форма

Веб-приложение для обработки платежей с многошаговой формой.

## Описание

Интерактивная платежная форма с пошаговым процессом заполнения данных. Приложение построено на React и использует современный UI/UX дизайн.

## Технологии

- React 18
- HTML5
- CSS3
- Babel Standalone

## Использование

Откройте файл `index.html` в браузере или разместите его на веб-сервере.
